# Papers

This directory contains 22 papers relevant to the research topic "Topological Persistence Analysis of LLM Training Dynamics for Scaling Prediction."

## Organization

- **PDFs**: All papers are stored as PDFs with descriptive filenames
- **Deep reading notes**: `notes_*.md` files contain detailed extraction from key papers
- **Chunked PDFs**: `pages/` subdirectory contains papers split into 3-page chunks for systematic reading
